## About project

Link: https://col-todo-list.vercel.app

Technologies: React, TypeScript, MobX.

## Installation

```
npm install
```

#### Before commits

```
npm run prepare
```

### Run project

#### Dev

```
npm run start:dev
```

#### Prod

```
  npm run build
  npm run start:prod
```

## Or run in docker

```
docker-compose up
```
